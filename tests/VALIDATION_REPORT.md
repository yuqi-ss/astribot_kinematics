## 关键结论

1. **FK 精度**: Pinocchio 实现与 MuJoCo 对同一 URDF 的解析几乎完全一致（误差 ~1e-12 mm），与真实 HDF5 数据的亚毫米级误差主要来自录制数据的时间戳抖动。

2. **IK 稳定性**: 两步求解（torso → arms）在全帧验证下保持 100% 收敛率，所有帧均满足 `pos<5mm` 和 `rot<1deg` 阈值。

3. **闭环验证**: `run_ik_hdf5.py`（项目 FK 回代）与 `sim_mujoco_ik.py`（MuJoCo 闭环）给出的误差统计完全一致，证明 IK 解在独立仿真器中可复现，不是内部自洽假象。

4. **工程可用性**: 单帧 IK 耗时 ~0.15 ms，吞吐 >6k frames/s，满足实时轨迹生成需求。
